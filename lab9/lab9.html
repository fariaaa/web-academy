What is inline CSS?

Inline CSS is putting a CSS file into HTML file instead of continuing on with an extended CSS. inline stylesheet is basically when we write css styles in html tags using attribute “style”. Inline CSS helps with incorporating style rules to spcific elements of HTML.

What is Internal or attached CSS?

An internal CSS is a CSS style used for for only one HTML file. Internal stylesheet is when we write code in between < style > and < /style > tag in head section of html document An internal stylesheet holds CSS rules for the page inside the head section of the HTML file.

What is external CSS?

External CSS is a seperaste CSS file. It can be used when you want to create several changes across mltiple pages.The CSS extension should be linked with the HTML file by using the link tag. It is defined in < head > section of the HTML page inside of the < style > tag
