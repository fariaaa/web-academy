tables and colors
by faria hossain

In order to create a table you have to start with "tr" including an opening and ending, add th with "<>" and then add your categories. After, add td w/"<>" and include the text you want in the boxes, make sure to end it with "< / t h>. In order to create another row you will have to add another"tr" with "<>".
