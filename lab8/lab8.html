CSS
By Faria Hossain

CSS is a language used in webpages to help with creating look and layout. It helps with the fonts, color, text, spacing, etc. I used CSS in my webpage for the incorporation of my index/home page. I created a header and navigation bar by using the codes, ".header{ background-color: rgb(32, 32, 32); padding: 25px; text-align: right; color: #FFAFDD;}". These codes helped with color, alignment, and padding. CSS and HTML are both quite similar. HTML contains the content while CSS contains the design of the content. They both help eachother.
